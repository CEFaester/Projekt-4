# Projekt-4
Dette projekt har til formål at udvikle en interaktiv online prototype, som skal øge opmærksomheden omkring SIND blandt unge i alderen 20–25 år. Prototypen skal være engagerende, informativ og brugervenlig, med fokus på frivilligt arbejde og mental sundhed.

Dette projekt er et fiktivt projekt udviklet som led i vores læringsforløb.
